#使用したもの

Java
NetBeans(フレームワーク)

#説明
Apache HTTP サーバのアクセスログを解析するプログラム

（問1）ログを読み込み、all_access_logに一行ずつ格納。
　　　取得した文字列をspiltを使用して切り出し、時間(time_log)・ホスト(host_log)ごとに分別しアクセス回数を数えて出力。
　　　ホストごとの集計は連想配列を使用し回数の多い順にソートした。

（問2）入力する複数のログのパスをfilelistにまとめ、作成したFileListsを使用して
　　　stream形式にしてmainlistに格納しファイルの読み込みを行った。

（問3）年月日を8桁の整数に置き換え、ログの集計時に比較。（2019年12月1日→20191201）
　　　この比較は入力されたログの日付が指定された日付の数値以内に存在すれば集計される。
　　　この時に指定された期間内のログのみを集計する。

（問4）一度に2GB（4,000,000行）を読み込むことが可能として作成。
　　　（3行で約500byte、2GBで2,147,483,648byte、割ると約4,294,967行、少なく見積もり4,000,000行で想定した）
　　　count_gyouでカウントし400万行読み込んだらいったん表示の処理を行い、再び同じ操作を
　　　ファイルを読み終えるまで継続。読み終わったら、flag_byteが0になり、その時点で処理を行い結果を表示する。

#必要条件
Windows環境で作成した。
java version - 1.8.0_161

#使い方
時間・ホスト毎のアクセス回数確認：　　57行～61行目でファイルを指定し格納、実行することで確認可能。
複数のファイルの読み込み　　　　：　　57行～61行目で使用したいファイルをfilename1から順に入れていくことで読み込み可能。
　　　　　　　　　　　　　　　　　　　64行目にもfilename~を追加してください。
期間の指定　　　　　　　　　　　：　　44，45行目でstart_dateからend_dateまでの日付の指定を整数8桁で入力することで実行可能。
　　　　　　　　　　　　　　　　　　　start_date、end_dateの両方に0を入れることで期間指定無しの判定になる。
